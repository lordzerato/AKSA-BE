# AKSA-BE

&nbsp;

## Description
This app is used to manage Books and their Author.

## Fetures
- Rest API
- Search

## How to use server
1. Install Node.js
2. Open project in code editor
3. Open terminal
4. Open directory Server in terminal (cd Server)
5. npm install
6. npm i -D nodemon sequelize
7. npm run dbc
8. npm run dbm
9. npm run dbs
10. npm run start
